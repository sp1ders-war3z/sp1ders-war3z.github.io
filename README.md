# sp1ders war3z

Cyberpunk hardware archive — GitHub Pages site.

## Deploy

1. Push this repo to `sp1derswar3z.github.io` (or enable Pages in repo Settings → Pages → branch `main` / root).
2. Site goes live at `https://sp1derswar3z.github.io`

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`
